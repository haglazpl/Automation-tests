## Setup - Automation tests

To run this project:

* Run Git Bash.
* Choose the project location in Git Bash.
* Type $ git clone https://github.com/haglazpl/Automation-tests
* Press enter.
* Now run IntelliJ PyCharm.
* Open the project location in File Tab.
